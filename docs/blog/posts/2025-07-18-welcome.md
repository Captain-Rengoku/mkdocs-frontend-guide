# 👋 Welcome to My Frontend Guide

> Fast, simple, and beautiful documentation — powered by **MkDocs + Material**.

This is my living, open notebook for learning modern frontend development with
**React**, **TypeScript**, and the surrounding ecosystem. I write these pages as I
learn, so they double as a personal reference and (hopefully) a useful resource
for anyone walking the same path.

<!-- more -->

---

## 📚 What you'll find here

- **React Docs** — components, state, hooks, forms, routing, and state management
  (Redux Toolkit, Zustand).
- **API Integration** — Fetch & Axios, authentication, TanStack Query, real-time
  data, security, and file handling.
- **TypeScript Docs** — from the basics through generics, advanced types, and
  React + TypeScript patterns.

Use the tabs in the top navigation to jump between sections, and the search box
(press ++f++ or ++s++) to find anything quickly.
